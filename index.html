📖 同時翻訳マスター 操作説明書

1. 使用手順

  1. 設定の確認: 上部バーで「音声（話す側）」と「翻訳（表示側）」を選択します。
  2. 開始: 「同時翻訳開始」ボタンを押し、マイクを許可します。
  3. 対話: 画面に向かって話すと、リアルタイムで字幕が表示されます。
  4. 終了: 「停止」ボタンで記録を自動保存し、セッションを終了します。

2. 動作環境と安定性

  • 推奨ブラウザ: Google Chrome または Microsoft Edge。※SafariやFirefoxでは音声認識が動作しません。
  • ネット環境: 音声解析と翻訳はGoogleのサーバーで行うため、安定したWi-Fi環境が必要です。
  • API制限: 短時間の大量リクエストにより翻訳が止まる場合があります。一文を丁寧に区切って話すと安定します。

3. プライバシーと機密保持

  • 通信の安全: 通信はHTTPSで暗号化されており、道中で第三者が内容を盗み見ることはできません。
  • データの扱い: 音声およびテキストは処理のためGoogleのサーバーへ送信されます。Google翻訳のWebサイト利用時と同様の機密レベルとなります。
  • 利用上の注意: パスワードや極めて機密性の高い個人情報のやり取りには使用しないでください。

4. 困った時の対応策

  • 翻訳が更新されない: 一度「停止」し、ブラウザを再読み込みしてください。改善しない場合は数分置いて再開するか、別のブラウザを試してください。
  • 音声認識が反応しない: マイクの接続とブラウザの「マイク許可」設定を再確認してください。

5. データの活用

  • Excel連携: 保存されたテキストをコピーし、Excel等のA1セルに貼り付けるだけで「時間・原文・翻訳」が綺麗に列分けされます。
